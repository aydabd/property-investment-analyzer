# 02-Plot Analysis – Ortoceratiten 1-2, Södra Råbylund

**Datum:** 2026-05-23

## Rekommenderat byggkoncept

**Friliggande enbostadshus i 2 plan** bedöms vara det mest rimliga och planmässigt säkra konceptet för tomten.

## Preliminär volymbedömning

| Parameter | Bedömning |
| --- | --- |
| Rekommenderad BOA | **150–165 kvm** |
| Estimerad BTA | **185–205 kvm** |
| Våningar | Troligen **2 plan** |
| Kommersiell optimering | Bygg för marknadsmatchning, inte maximal möjlig byggrätt |

## Tomtens styrkor

- Stor tomt: **826 kvm**
- Attraktivt nyproduktionsläge i Lund
- God sannolikhet att ett normalt fribyggarhus får plats utan trång situation
- Rimlig möjlighet till parkering, trädgård och komplementytor

## Osäkerheter som återstår

Flera plan- och anslutningsposter kunde inte live-verifieras i denna körning. Dessa måste bekräftas innan bindande beslut:

- Exakt detaljplan och tillåten byggrätt
- Eventuellt BOA-/BTA-tak
- Prickmark, avstånd till gräns, byggnadshöjd och takkrav
- Exakta kommunala avgifter och anslutningar

## Kostnadsrelevanta tomtposter

| Post | Intervall |
| --- | ---: |
| Kommunala kostnader / anslutningar | **325 000–550 000 kr** |
| Särskilda tomt-/planrelaterade kostnader | **175 000–740 000 kr** |
| Geoteknisk risk | **Medel** |

## Slutsats

Tomten får ett **VILLKORLIGT GO på tomtnivå**. Fysisk och kommersiell potential finns, men analysen måste vara konservativ eftersom detaljplan, anslutningar och geoteknik ännu inte är helt verifierade.
