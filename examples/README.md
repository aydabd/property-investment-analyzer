## Hur du kan använda exemplen

### För inspiration

Läs igenom exemplen för att se vilken detaljnivå agenterna levererar.

### För validering

Kör ett exempel genom systemet för att verifiera att allt fungerar:

```bash
# Skapa ny issue med samma data som 01-stangby-beryllen
# Jämför resultatet med det sparade exemplet
```

### För utbildning

Nya partners kan läsa exemplen för att förstå systemet innan de skapar sin egen issue.

## Bidra med exempel

När du kör en riktig analys som blir bra:

1. Anonymisera personnummer och känsliga namn
2. Kopiera `final-report.md` till `examples/NN-kort-beskrivning.md`
3. Öppna PR
4. Andra kan då lära från ditt projekt

## Viktiga förbehåll

Exempel är **historiska snapshots** – marknadsdata i dem kan vara föråldrad.
Kör alltid en egen analys för bindande beslut.
